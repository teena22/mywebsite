Imaginit

Here we proposed a system which removes noise from the image for better view. User can save image with different effects and can also see his/her previous edited image. System also provides editing effects like resize ,rotate and flip the image. Now-a-days we are using so many image editors. But the problem with normal current image editors is that we can’t get noise resolved image. We are making a image editor in such a way that it will make a image noise resolved and will also provide some existing features. It will save the image with all the changes and reduce its size.
